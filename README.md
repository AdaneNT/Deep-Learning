# Deep Learning Lecture Materials

This repository contains lecture materials developed for university-level courses in Deep Learning.

## Overview

The materials were developed for postgraduate teaching, focusing on structured explanation of key deep learning concepts and models.

## Contents

Lecture slides covering topics such as:
- Neural networks
- Backpropagation
- Convolutional neural networks (CNNs)
- Deep learning architectures

## Teaching Approach

The materials are designed to:
- Provide clear conceptual explanations
- Support classroom teaching and discussion
- Serve as a foundation for practical and project-based learning

## Notes

- These materials are intended for educational use
- Some content may be adapted for teaching purposes
